# ai-supermarket

这是一个名为 ai-supermarket 的 AI 模板超市 Web 应用项目，基于 Vue 3 + TypeScript + Vite + Tailwind CSS 技术栈构建。项目采用前后端一体化架构，后端使用 Express + PostgreSQL 数据库，集成 bcrypt 和 jsonwebtoken 实现用户认证。

用户端功能 包括：首页展示、模板浏览与详情查看、购物车管理、订单系统、用户中心（收藏、下载、设置、支持）等； 管理端功能 涵盖：仪表盘数据展示、用户管理、订单管理、模板管理、分类管理、Banner管理、FAQ管理、站点设置和系统设置等模块。整体是一个完整的模板交易平台，支持用户注册登录、模板购买下载以及管理员对平台内容的全面管理。
